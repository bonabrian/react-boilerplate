# About
React js template to build a new project.

## Quick start
1. Clone this repo using:
```bash
git clone --depth=1 https://github.com/bonabrian/react-boilerplate.git
```

2. Install dependencies.
```bash
cd react-boilerplate
npm install
npm start
```
